# Car Price Prediction

Machine Learning project to predict car prices using:

- Python
- Pandas
- NumPy
- Scikit-Learn
- Google Colab

## Steps
1. Data Loading
2. EDA
3. Data Preprocessing
4. Feature Engineering
5. Model Training
6. Evaluation

## Dataset Features
- Car Make
- Car Model
- Year
- Engine Size
- Horsepower
- Torque
- 0-60 MPH Time

Target:
- Price (USD)
